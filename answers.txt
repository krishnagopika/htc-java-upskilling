Q2. :Write a program to find if a matrix is symmetric. Print Yes if the matrix is symmetric, else print No.


import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
        Scanner k = new Scanner(System.in);
        int m = k.nextInt();
        int n = k.nextInt();
        int[][] p = new int[m][n];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                p[i][j] = k.nextInt();
            }
        }
        boolean isSymmetric = true;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (p[i][j] != p[j][i]) {
                    isSymmetric = false;
                    break;
                }
            }
            if (!isSymmetric) {
                break;
            }
        }
        if (isSymmetric) {
            System.out.println("Yes");
        } else {
            System.out.println("No");
        }
    }
}

Q4: John and Jacob are playing a dice game, if the number rolled is even, John get's a point., if the number rolled is odd Jacob get's a point. John and Jacob played N games. Print who won at the end of N games. if its a tie print Tie.
Simple
Sample Input: 4 7 2 3 4
Sample Output:
Tie
Sample Input: 4 9 2 3 5
Sample Output:
Jacob
Sample Input: 4 3 2 6 4
Sample Output:
John
ANS

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner k = new Scanner(System.in);
        int N = 5; // Number of games
        int[] a = new int[N];
        int johnScore = 0;
        int jacobScore = 0;
        
        for (int i = 0; i < N; i++) {
            int m = k.nextInt();
            a[i] = m;
        }
        
        for (int j = 0; j < N; j++) {
            if (a[j] % 2 == 0) {
                johnScore += 1;
            } else {
                jacobScore += 1;
            }
        }
        
        if (johnScore == jacobScore) {
            System.out.println("Tie");
        } else if (johnScore > jacobScore) {
            System.out.println("John");
        } else {
            System.out.println("Jacob");
        }
    }
}

import java.util.Scanner;

public class CaesarCipherDecryption {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter the encrypted string: ");
        String encryptedString = scanner.nextLine();

        System.out.print("Enter the key: ");
        int key = scanner.nextInt();

        String decryptedString = decrypt(encryptedString, key);
        System.out.println("Decrypted message: " + decryptedString);

        scanner.close();
    }

    public static String decrypt(String encrypted, int key) {
        StringBuilder decrypted = new StringBuilder();
        for (int i = 0; i < encrypted.length(); i++) {
            char currentChar = encrypted.charAt(i);

            if (Character.isLetter(currentChar)) {
                char base = Character.isUpperCase(currentChar) ? 'A' : 'a';
                char decryptedChar = (char) (base + (currentChar - base - key + 26) % 26);
                decrypted.append(decryptedChar);
            } else {
                decrypted.append(currentChar);
            }
        }
        return decrypted.toString();
    }
}

The XYZ company has N number of employees. If an employee's name is the least common, that name is rare. Find the rare name in XYZ company(print only one name).
Complex
Note:Not suitable for C or C++.
Input format:
N
Name1 Name 2 Name 3 ....,Name n
Sample Input:
8
Jake Jane Jake Albert Alex Albert Albert Jane
Sample Output:
Alex
import java.util.*;

public class RareNameFinder {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        sc.nextLine(); // Consume the newline character
        
        String names = sc.nextLine();
        String[] namesArray = names.split(" ");
        HashMap<String, Integer> hm = new HashMap<>();
        
        for (String i : namesArray) {
            if (hm.containsKey(i)) {
                hm.put(i, hm.get(i) + 1);
            } else {
                hm.put(i, 1);
            }
        }
        
        int count = Integer.MAX_VALUE;
        String uniqueName = "";
        
        for (Map.Entry<String, Integer> entry : hm.entrySet()) {
            if (entry.getValue() < count) {
                count = entry.getValue();
                uniqueName = entry.getKey();
            }
        }
        
        System.out.println("Rare name is " + uniqueName);
    }
}
